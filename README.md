# Banking System

A Java console application that simulates basic banking operations
like deposit, withdrawal, balance check, and transaction history.

## Features
- Deposit money into account
- Withdraw money with balance validation
- Check current balance
- View complete transaction history
- Menu-driven interface — runs until user exits

## How to Run
1. Download `Account.java` and `Main.java`
2. Compile: `javac Main.java Account.java`
3. Run: `java Main`
4. Choose options from the menu (1-5)

## Sample Interaction
1. Deposit
2. Withdraw
3. Check Balance
4. Show History
5. Exit
